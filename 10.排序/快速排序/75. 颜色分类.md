# 75. 颜色分类
给定一个包含红色、白色和蓝色，一共 n 个元素的数组，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。  

此题中，我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。  

##### 示例 1：  
  
输入：nums = [2,0,2,1,1,0]  
输出：[0,0,1,1,2,2]  
##### 示例 2：  
  
输入：nums = [2,0,1]  
输出：[0,1,2]  
##### 示例 3：

输入：nums = [0]  
输出：[0]  

### 方法1：快速排序
显然，本题可以使用三路快排的思想，将数组分为三部分：0、1、2
```
class Solution {
void sortColors(vector<int>& nums) {
    // [0, l]   : 0
    // [l+1, i) : 1
    // (r,  nums.size()-1]:2
    int l = -1;
    int r = nums.size()-1;
    int i = 0;
    
    while(i<=r)
    {
        if(nums[i] == 0)
        {
            swap(nums[l+1], nums[i]);
            l++;
            i++;
        }
        else if(nums[i] == 1)
        {
            i++;
        }
        else{
            swap(nums[i], nums[r]);
            r--;
        }
    }
}
```
